Project Structure

Project Structure

Typical NodeJs project structure, where all files in "src" folder except package.json and node_modules...etc.

-- index.js the starting point which contain only imports and initialization code.

-- all code executions starts from route files (./modules/routes)

    •  index.jsthe starting point of the project
    •  modules/include all the project modules
      •  configurations/include the configuration modules
        •  cert.pemthe https server certificate
        •  key.pemthe https server private key
        •  cors.jsthe cors options
        •  custom-middleware.jsall the custom middleware used by the project
        •  https-server.jsthe https server configurations
        •  jwt-generator.jsthe jwt token methods
        •  mzh-tools.jshandy general methods
        •  server-status.jscontain methods to check if the server is connected or having connection issues
      •  mongodbcontain all the mongodb operations & models
        •  connect.jscontain initialization of mongodb connection
        •  tokens.jscontain all the tokens mongodb schema & methods
        •  users.jscontain all the users mongodb schema & methods
      •  passportcontain all passport configuration to signing locally or via social platforms
        •  passport.jscontain passport general configurations
        •  facebook.jscontain facebook passport strategy
        •  google.jscontain google passport strategy
      •  routescontain all the express routers
        •  signing.jscontain the local passport signing routers
        •  facebook.jscontain facebook passport routers
        •  google.jscontain google passport routers
